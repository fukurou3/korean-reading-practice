# 1. 은/는（〜は）

順番：私 → 友達 → 先生 → 学生 → 人

| 韓国語         | 日本語      |
| ----------- | -------- |
| 저는 학생이에요.   | 私は学生です。  |
| 친구는 학생이에요.  | 友達は学生です。 |
| 선생님은 학생이에요. | 先生は学生です。 |
| 학생은 학생이에요.  | 学生は学生です。 |
| 사람은 학생이에요.  | 人は学生です。  |

※「先生は学生です」は意味としては変だけど、**今は文法パターンを身体に入れる練習**なのでOK。自然さより型の固定を優先。

---

# 2. 이/가（〜が）

順番：私 → 友達 → 先生 → 学生 → 人

| 韓国語      | 日本語     |
| -------- | ------- |
| 제가 와요.   | 私が来ます。  |
| 친구가 와요.  | 友達が来ます。 |
| 선생님이 와요. | 先生が来ます。 |
| 학생이 와요.  | 学生が来ます。 |
| 사람이 와요.  | 人が来ます。  |

※「저가」ではなく、会話では普通 **제가** を使う。

---

# 3. 을/를（〜を）

順番：本 → 水 → 食べ物 → 映画 → 音楽

| 韓国語     | 日本語      |
| ------- | -------- |
| 책을 봐요.  | 本を見ます。   |
| 물을 봐요.  | 水を見ます。   |
| 음식을 봐요. | 食べ物を見ます。 |
| 영화를 봐요. | 映画を見ます。  |
| 음악을 봐요. | 音楽を見ます。  |

※「音楽を見る」は不自然。だけど、ここでは **을/를 の形を固定する教材**として割り切る。自然な例にすると動詞がバラけて、学習効率が落ちる。
